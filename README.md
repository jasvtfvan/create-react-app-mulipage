# create-react-app-mulipage
A simple multiple pages project for react, rewrite create-react-app, based on npm run eject of CRA
<br />
# yarn or npm to install
```
yarn install (or npm install)
```
<br />
# run
```
npm start
```
# build
```
npm run build
```
